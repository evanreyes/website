Here are two projects I worked on at Coding Dojo. They are Django-based apps that highlight my database work and showcase one-to-one, one-to-many, and many-to-many data relationships. These apps feature a login and registration component that performs checks on user input and rejects login or registration if certain validations are not met.

These apps are quite basic visually, with minimal CSS applied, but do use Bootstrap for their structure. Go ahead and test them out by registering your credentials (a valid email address is not necessary) and playing around with the different forms and pages. Also check out my Github to see the underlying code. Both apps utilize session variables to determine the currently logged-in user, and make database queries and calls based on that information. Templates update according to who is logged in. You can check out these apps' source code on Github.




TRAVEL BUDDY

Travel Buddy features login and registration, and allows users to book "travel itineraries" for future dates. Users can "join" other users' trips to create lists of individuals who are traveling together.



BOOK REVIEWS

Book Reviews uses a login and registration component, and allows logged-in users to postbook reviews with ratings. My app does not allow for duplicate authors if a user has already added them to the database. Through the various templates, you can see a user's reviews, an author's books, and more.
